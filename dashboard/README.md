# مركز الأزمات والطوارئ — وزارة الخارجية

Saudi Ministry of Foreign Affairs — Global Risk Intelligence Dashboard.

A Vite + React + TypeScript PWA that tracks live global risk events (earthquakes,
storms, conflict, disease, etc.) and Saudi travelers abroad, with real-time alerting
and a committee-response view.

## Development

```bash
npm install
npm run dev
```

Copy `.env.example` to `.env` and fill in `VITE_TWELVEDATA_API_KEY` to enable the
Economic Watch widget's live gold/oil prices. Everything else works with no keys.

## Build

```bash
npm run build
```
